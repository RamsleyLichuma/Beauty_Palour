How to run the Beauty Palour Management System (The Vera Beuty Palour) Project

1. Download the zip file

2. Extract the file and copy The Vera Beuty Palour folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name The Vera Beuty Palourdb

6. Import The Vera Beuty Palourdb.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/The Vera Beuty Palour (frontend)
